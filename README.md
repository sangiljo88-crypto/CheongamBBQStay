# 청암스테이 — 공장직결 뒷고기 스테이 랜딩페이지

## 프로젝트 개요
새림 공장 직결 뒷고기 바베큐 + 숙박 코스형 상품을 판매하는 게스트하우스 랜딩페이지입니다.

## 현재 완료된 기능

### 페이지 구조 (섹션 순서)
1. **HERO** — 공장직결 뒷고기 스테이 메인 슬로건 + 가격 (35,000원~) + CTA 버튼
2. **INFO BAR** — 핵심 키워드 바 (최소2인, 숙박포함, 뒷고기 바베큐, 단체, 통대관)
3. **GALLERY** — 3컬럼 사진 스트립 (바베큐존, 뒷고기, 라운지)
4. **WHY (문제 공감)** — 고기 모임의 번거로움 4가지
5. **SOLUTION** — 청암스테이의 해결책 설명
6. **INCLUDED (상품 요약)** — 코스 포함사항 6가지 아이콘
7. **COURSE ⭐** — 3개 코스 카드 (라이트 35k / 시그니처 55k / 블랙 75k)
   - 시그니처: **BEST 왕관 배지** (반짝이는 sparkle 애니메이션)
   - 블랙: **HIT 배지** (pulse 애니메이션)
8. **MENU BOARD** — Tally 코스 상세 메뉴판 (레스토랑 메뉴판 스타일)
   - 3개 코스별 포함 항목 상세
   - 할인 안내 (미취학 무료, 학생 할인)
   - 추가 옵션 안내
9. **DISCOUNT & REVIEW PROMO** — 프로모션 카드 2개
   - 단체 할인: 10인 이상 10%, 30인 이상 20%
   - 네이버 리뷰 보상: 프리미엄 뒷고기 모듬 300g 1팩 (약 1만원 상당)
10. **MID CTA** — 중간 상담 요청 버튼
11. **SPACE** — 공간 사진 기반 소개 (라운지, 객실, 카페, 샤워실)
12. **ROOMS** — 인원별 객실 배정 안내
13. **USP (공장직결)** — 일반유통 vs 공장직결 비교
14. **GROUP** — 단체/워크숍/통대관 가격 및 구성
15. **BUDGET** — 이용 예산 예시 (2인/4인/10인)
16. **FIT** — 추천 대상 4가지
17. **FAQ** — 9개 질문답변 (아코디언)
18. **FINAL CTA** — 최종 예약 유도
19. **FOOTER** — 기본 정보

### 기술 구현
- Tally 팝업 연동: `https://tally.so/r/44LPQr` (embed.js 스크립트 포함)
- GSAP + ScrollTrigger 스크롤 애니메이션
- BEST 배지: 금색 그라데이션 + 왕관 아이콘 + sparkle 파티클 애니메이션
- HIT 배지: 빨강-오렌지 그라데이션 + 번개 아이콘 + pulse 애니메이션
- 완전 반응형 (768px, 480px 브레이크포인트)
- 외부 이미지 URL 사용 (Base64 제거)
- Google Fonts (Noto Serif KR + Noto Sans KR)
- Font Awesome 6.5.1 아이콘
